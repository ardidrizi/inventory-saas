Build a production-ready fullstack SaaS Inventory and Order Management System in ONE repository.

Tech stack:

- Backend: Node.js, Express, TypeScript, MongoDB (Mongoose), JWT auth, Zod validation
- Frontend: React, TypeScript, React Router, Axios, Recharts
- Clean scalable architecture
- Role-based authentication (admin, manager)
- Product CRUD with soft delete
- Order creation with automatic stock reduction
- Dashboard analytics
- Proper error handling
- Production-ready structure

Follow clean architecture principles:

- Controllers handle request/response
- Services contain business logic
- Models define schema
- Middleware handles auth and errors
- Frontend uses services layer for API calls
